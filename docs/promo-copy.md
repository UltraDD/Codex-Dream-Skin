# Codex Dream Skin 宣传文案

## 一句话

给 Codex 桌面端换一张会呼吸的脸：背景可换、控件可用、随时恢复。

## 简短版本

我把 Codex Dream Skin 开源了。

它不是把整张假 UI 贴上去，而是在本机通过 CDP 注入主题层：侧栏、卡片、项目选择和输入框仍是原生控件。macOS 与 Windows 都支持导入自己的纯背景、保存主题、检查效果和恢复官方外观。

公开仓库只分发程序化抽象预设和脱敏示例，不包含真人、角色、客户品牌或私人任务截图。

## 长版本

Codex Dream Skin 是一套面向 Codex 桌面端的外部主题工具。它不修改官方安装包、`app.asar` 或代码签名，而是在本机回环 CDP 会话中注入背景和样式。

核心能力：

- macOS / Windows 双平台安装入口。
- 导入自己的宽幅纯背景。
- 保存、切换和暂停本地主题。
- 首页与任务页使用不同的背景强度。
- 一键 Restore 官方外观。
- 完整的验证、日志和故障排查路径。

素材边界：只公开你拥有或明确获准再分发的内容；发布截图前移除账号名、聊天、真实任务标题和本机路径。
