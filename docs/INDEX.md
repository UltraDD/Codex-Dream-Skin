# 文档导航

如果你第一次来到这个仓库，请从下面第一篇开始。这个 Fork 的目标是帮助你把自己的图片做成一套可安装、可验证、可恢复的 Codex 皮肤，而不是分发某个角色或人物的成品主题包。

## 从这里开始

- [尽量一次成功：自定义 Codex 皮肤工作流](./one-shot-skin-workflow.md)：先区分皮肤、效果图、悬浮组件与可发行预设，再用统一契约完成制作、验收和恢复。
- [从一张图做出自己的 Codex 皮肤（Windows）](./windows-custom-skin-guide.md)：从准备背景图、安装、换图、验收到排错的完整流程。
- [平台能力与配置字段](./platforms.md)：macOS / Windows 差异，以及 `safeArea`、`taskMode` 等进阶参数。
- [参考图生图提示词](./reference-background-prompt-guide.md)：怎样生成不带假 UI、适合真实界面覆盖的背景图。
