# Codex Dream Skin Reference Image Guide

Reference images should guide composition, palette, and atmosphere only. They should not transfer UI, text, a person's identity, protected characters, or branding into a new background.

## Confirm rights first

Use only:

- Images you created and may redistribute.
- Media with explicit permission for modification and public redistribution.
- Abstract presets in this repository that are identified as programmatically generated.

Do not use celebrity likenesses, film or animation characters, customer material, private chats, or screenshots containing accounts and local paths as public-release references.

## Assign reference roles

| Reference | Extract only | Do not copy |
|---|---|---|
| Image 1 | Spatial composition and negative space | Text, logos, controls |
| Image 2 | Color, lighting, and material language | Brand identity or character design |
| Image 3 (optional) | Position of a subject you are authorized to use | Any unauthorized identity or likeness |

## General template

```text
Use the references only for composition, color palette, lighting, and material
language. Create a new wide background image for a desktop workspace. Keep the
left third and lower center calm for native controls. Concentrate detail toward
the right edge. Do not reproduce any person, character, logo, signature,
readable text, sidebar, card, button, input box, window frame, or other UI from
the references. Produce background art only, with an original composition and
no identifying private content.
```

## No-person template

```text
Transform the reference mood into an original abstract environment. Preserve
only the broad palette and lighting rhythm. Remove all people, characters,
brands, text, and interface elements. Use soft depth, restrained detail, and
wide negative space suitable for a native desktop application overlay.
```

## Acceptance before import

- No readable text, fake controls, or window chrome.
- No celebrity, protected character, brand, or private task content.
- Enough calm space for the sidebar and composer.
- Readable on both home and task routes.
- Traceable source, permission, and generation process.

If any check fails, do not publish the image as a preset. Revise the prompt and generate again.
