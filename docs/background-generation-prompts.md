# Codex Dream Skin 纯背景生图提示词

目标是生成可被真实 Codex 控件覆盖的纯背景，而不是一张带假界面的效果图。

## 通用要求

- 宽幅构图，建议 16:9 或 16:10。
- 主要视觉焦点放在右侧或边缘，为左侧导航和中下部输入框留出安静区域。
- 不生成侧栏、卡片、按钮、输入框、菜单、窗口边框、Logo 或可读文字。
- 不使用真人肖像、受保护角色、客户品牌或未经授权的参考图。
- 暗色和浅色背景都需要保留足够的文字对比度。

## 无人物抽象背景

```text
Create a clean ultra-wide desktop application background, abstract atmospheric
light, layered depth, restrained geometric forms, subtle cinematic glow,
visual focus on the right third, calm negative space on the left and lower
center for native application controls, premium but understated, no people,
no characters, no logos, no readable text, no sidebar, no cards, no buttons,
no input box, no browser chrome, no mock interface, background art only.
```

## 自然景观背景

```text
Create a wide cinematic landscape background for a desktop workspace, soft
mist, distant mountains and water, gentle atmospheric perspective, detail
concentrated near the right edge, quiet negative space on the left and lower
center, elegant restrained palette, no people, no buildings with signage,
no logos, no readable text, no interface elements, background art only.
```

## 科幻城市背景

```text
Create an original wide futuristic cityscape at dawn, luminous architecture,
clean horizon, subtle volumetric light, focal structure on the right, generous
empty space for native controls, no recognizable franchise design, no people
in close-up, no logos, no readable text, no interface, background art only.
```

## 验收清单

1. 图中没有任何 UI 或可读文字。
2. 没有真人、受保护角色、品牌标志或可识别私人内容。
3. 左侧与输入框区域不会被高对比细节干扰。
4. 在首页和任务页都能保持可读性。
5. 图片来源和再分发权限可说明、可追溯。

通过后再按[自定义皮肤工作流](./one-shot-skin-workflow.md)导入和验证。
