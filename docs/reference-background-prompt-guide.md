# Codex Dream Skin 参考图生图指南

参考图只负责构图、配色和氛围，不应把原图中的 UI、文字、人物身份或品牌复制进新背景。

## 先确认权利

仅使用以下来源：

- 你自己创作并拥有再分发权的图片。
- 明确允许修改与公开再分发的素材。
- 仓库内标明为程序化生成的抽象预设。

不要把真人肖像、影视动漫角色、客户物料、私人聊天截图或带账号/路径的界面截图作为公共发行参考。

## 固定参考图职责

| 参考 | 只提取 | 不复制 |
|---|---|---|
| Image 1 | 空间结构、留白位置 | 文字、Logo、控件 |
| Image 2 | 色彩、光线、材质 | 品牌识别、角色造型 |
| Image 3（可选） | 已授权主体的大致位置 | 未授权身份或肖像 |

## 通用模板

```text
Use the references only for composition, color palette, lighting, and material
language. Create a new wide background image for a desktop workspace. Keep the
left third and lower center calm for native controls. Concentrate detail toward
the right edge. Do not reproduce any person, character, logo, signature,
readable text, sidebar, card, button, input box, window frame, or other UI from
the references. Produce background art only, with an original composition and
no identifying private content.
```

## 无人物模板

```text
Transform the reference mood into an original abstract environment. Preserve
only the broad palette and lighting rhythm. Remove all people, characters,
brands, text, and interface elements. Use soft depth, restrained detail, and
wide negative space suitable for a native desktop application overlay.
```

## 导入前验收

- 没有可读文字、假控件或窗口边框。
- 没有真人、角色、品牌或私人任务内容。
- 左侧导航和输入框区域有足够留白。
- 图片在首页与任务页都不影响可读性。
- 来源、授权与生成过程可追溯。

任一项失败都不要进入公共预设；修正提示词后重新生成。
